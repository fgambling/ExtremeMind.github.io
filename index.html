Automatic Scoring System

Overview

A full‑stack web application that streamlines exam grading through AI‑assisted answer evaluation, configurable marking rules, and role‑based access control, featuring manual review tools, Excel data integration, and detailed reporting for admins, test developers, and markers. The system is built with modern cloud-native architecture and deployed on AWS for high availability and scalability.

Core Features
  • Automated Scoring – Intelligent evaluation with flexible marking rules (case, punctuation, spelling, grammar)
  • Role‑Based Access – Dedicated dashboards for Admins, Test Developers, and Markers
  • Manual Review – Flagged answers for human validation and scoring
  • Excel Integration – Import student responses and export scored results
  • Real‑Time Reporting – Live progress tracking and performance analytics
  • Advanced Linguistic Checks – Spelling & grammar analysis for accurate scoring
  • Cloud‑Native Architecture – Built for AWS deployment with auto-scaling capabilities

Relationship AI Agent Management Platform

Overview

  • Connection Square – Publish and discover opportunities with smart matching, contact management, and business card integration
